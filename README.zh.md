# Echo Player

Echo Player 是一个基于 WPF 的视频播放器，专为语言学习者打造。

### 功能
#### 字幕功能
- 点击单词即可查询翻译
- 利用 AI 进行句子分析
- 鼠标悬停显示字幕

#### 生词本
- 收藏单词和句子以便复习

#### 系统要求
- Windows 11

#### 快捷键
- 空格键：播放/暂停
- 左/右方向键：后退/前进
- 上/下方向键：音量控制
- Esc 键：退出全屏

#### API
- 牛津词典：单词查询
- OpenAI：句子分析
- 使用您自己的 API 密钥
- 或使用我们的免费 API 服务（目前免费）
